# C盘瘦身扫描助手

C盘瘦身扫描助手是一款 Windows 桌面小工具，用来帮助用户快速查看 C 盘空间占用情况，找出可能占用大量磁盘空间的文件、文件夹和缓存内容。

当 C 盘空间不足、电脑提示磁盘快满，或者系统运行变慢时，可以先用这个工具进行扫描，了解空间主要被哪些内容占用，再决定是否清理。

##主要用途

- 扫描 C 盘空间占用情况
- 帮助定位体积较大的文件和目录
- 辅助发现临时文件、缓存文件、下载残留、安装包等可清理内容
- 为手动清理 C 盘提供参考

## 使用方法

1. 下载本仓库中的 ``C盘瘦身扫描助手.exe``
2. 双击运行程序
3. 根据扫描结果查看 C 盘空间占用情况
4. 按需清理确认无用的文件

## 注意事项

清理 C 盘前，请先确认文件用途。不要随意删除不认识的系统文件、程序目录或重要个人文件。

本工具主要用于辅助扫描和判断，建议用户根据实际情况谨慎清理。
